一张图（最基本用法 markdown语法） ![Alt text](图片链接 "optional title") Alt text：图片的Alt标签，用来描述图片的关键词，可以不写。最初的本意是当图片因为某种原因不能被显示时而出现的替代文字，后来又被用于SEO，可以方便搜索引擎根据Alt text里面的关键词搜索到图片。 图片链接：可以是图片的本地地址或者是网址。“optional title”：鼠标悬置于图片上会出现的标题文字，可以不写。多图横向滚动 设计原理：将图片填充到表格的格子里，当表格超出页面显示的最大宽度，在表格的下方就会出现一个横向滑块。 不带表头的写法（css:table） <table> <tr> <td><img src="图片地址" width="图片宽度"/></td> <td><img src="图片地址" width="图片宽度"/></td> </tr> </table> 效果演示： 带表头的写法（markdown 语法） | 表头1 | 表头2 | 表头3 | 表头4 | 表头5 | | ---- | ---- | ---- | ---- | ---- | | 图片1 | 图片2 | 图片3 | 图片4 | 图片5 | 江滨公园1 江滨公园2 江滨公园3 江滨公园4 江滨公园5 缺点：不能自由控制图片显示的长度和宽度。……
