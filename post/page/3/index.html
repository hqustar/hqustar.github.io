使用hexo与github搭建博客网站

2017年2月12日
| 本站建设
参照Markdown 简单的世界并结合自己搭建过程中的点滴，将本站的搭建过程做个记录。 Hexo是一个开源的静态博客生成器，用node.js开发，作者是台湾大学生tommy351。 为什么是博客 对于个人网站来说，没有比博客更合适的形式了。在博客中，文章才是最主要的，一切都显得主次分明，干净利落。相比之下，论坛中主题和回复鱼龙混杂，阅读体验非常差。同时，博客比论坛的数据库小很多，便于维护。 为什么是静态博客 很多人选择在虚拟主机或vps上面搭建动态博客。但是这些主机商通常“免费的不稳定，稳定的不免费”。前一段时间，我观察了我的个人博客友链上面的几个站点，一部分在十几天之后就销声匿迹了。独立博客如此麻烦的维护工作，能不能减轻一些呢？正如阮一峰前辈所说，blogger分为三个阶段。最开始，是门户博客。之后，是独立博客。最后，觉得独立博客自己管理起来费劲，便找个别人来管的空间，自己负责写就好。如果我们能够找到这样的空间，在自己保留最大控制权前提下，由别人托管，会省去不少事情。 静态博客编译之后是纯html页面，优点就是支持它的环境十分好找，例如github、gitcafe、七牛云存储等站点都支持静态页面托管，自然是我们的首选了。由于github page在国内访问较慢，这篇文章用gitcafe做示范。gitcafe是天朝本地化的github，同样提供展示页和域名绑定功能，不需要备案，就是爽。 但是静态博客并非没有缺点。动态博客更新文章时，脚本是不变的，只需要更新数据库。静态博客要频繁改动文件，不支持增量式上传的东西，比如ftp，就难于管理。此外，还要十分熟悉git各种命令，才能部署页面。 准备工作 git node.js markdown编辑器 github 安装 git linux下面，在bash中键入： (Ubuntu, Debian) $ sudo apt-get install git (Fedora, Red Hat, CentOS) $ sudo yum install git windows或mac下，直接到git-scm官网下载安装。 安装 node.js linux下： $ sudo apt-get install nodejs $ sudo apt-get install npm yum同理。 windows或者mac下，直接到node.js官网下载安装。 windows还要设置环境变量，把node.js安装路径写进path里面，用半角分号分隔。 注册github账号 首先注册一个github账号username，之后创建一个username.github.io的项目。 如：我的github账号是hqustar，创建了一个hqustar.github.io项目。 安装和初始化 linux下打开bash，win下面打开cmd，输入： $ npm install hexo -g $ hexo init blog $ cd blog $ npm install $ hexo server 访问http://localhost:4000，会看到生成好的博客。……

那些独居中的点滴日常

1年1月1日
| 随笔
一个人独居可得一个闲字，就像丰子恺的《闲居》不以之为不幸，反而认为这是情趣上的适快。诚然，在群体生活中最难的就是保存个人的自由不被别人所窥探，独居仿佛在人与人的千丝万缕的混沌中生生开辟出了一块只属于自己的天地，留的住天性与纯真，不必顾虑他人的感受，逢迎他人的喜好。然而，祸福相依，独居固有其乐，也自当有其弊。独居的闲适稍微不留神也就变成了闲愁，观花花不语是愁，春水向东去是愁，人比黄花瘦也是愁，枯藤老树昏鸦是愁，大漠孤烟直是愁，潇潇的落木是愁……唉，要说闲愁最苦，也确有其理。记得去年这个时候，天气也是格外的闷热，每天夜里务必将门和窗都打开给房间通风，风扇一刻也不能停息，待到深夜1点左右，气温稍稍下降才能得以安寝。时间日久，我竟然觉得自己在夏天的时候精力更好，安之若素，处之泰然。可是今年夏天和以往不太一样了，连续的2星期，夜里9点左右我必然犯困，然后沾床就睡，睡也睡不长久，12点左右复醒。过了几小时稍有困意，又复睡去。睡眠被分成了两节，所以乘此间隙写成这篇文章，心中的任何感悟，也许是一句话，也许是一段话，不写出来那就都不复存在，无迹可寻。 说起独居很自然就想到“宅”，“独居"说的是一个人居住，中性词，一种客观的描述。”宅“指的是一个人不出去活动，选择自己躲在房间里，带有强烈的主观性。独居的人未必就喜欢宅，喜欢宅的人也未必就是独居。因为工作的缘故，我需在县城租房，父母在乡下仍有牵绊，不能与我日日相见，生活一起。我不是一个喜欢宅的人，我喜欢大自然的一切，风、雨、阳光、花草树木对我来说都是别有生机，每一天我都会带着我的小电驴去兜风或者在公园走上几圈，宅不能给人精神慰籍，而自然天生就有这股力量。树荫下晨读，石阶上捧书，鸟语花香同朗朗的书声一起在宣告”我才是大自然里最快乐的精灵“。 为了打发闲愁，总要找点事情做才是。散步运动方面：傍晚时分，饭前饭后走一走，走完北湖再去走文体操场，一天下来也总有5公里左右，运动量不算大，若能每天坚持下来，运动的效果得到巩固，也就足够了。学习提高方面：这里我必须说一句，并不是所有的东西都可以学以致用，学而无用的东西还少吗？在则，用以活学才能达到活学活用，像我这般看得都是闲书，且还看得飞快，除了与书同欢同悲之外，也没学到啥东西嘛！如果说有，倒是学会了矫情。哈哈，矫情的人儿爱写诗，会开心也会流泪。 说到吃，有人说这是天底下第一大难题，不亚于媳妇和母亲落水先救谁。工作日吃食堂，刚刚百度了一下，中国有八大菜系 ，如果说食堂的饭菜特点在鲜明一点点，那绝对可以排出九大菜系。好吧，我想说的是食堂饭菜菜色单一，容易吃腻。非工作日在外面吃就自由多啦，每天都可以吃主题料理。以牛为主题就有牛肉、牛肚、牛肉丸；以鸭为主题就有鸭脖子、鸭脯肉、鸭腿、鸭爪；以海鲜为主题就有贝壳、鱼、虾、蟹、软体动物；再搭配不同的时令蔬菜、辛辣酸甜，变幻万千，回味无穷，令人咂舌。好吧，说得我自己都不太相信… 这个夏天是独属于吃瓜群众的，我就是这其中的一员。按肤色分西瓜可以分成黑和白两种，按形状分可以分成长和圆两种，所以天下西瓜就有四种。怎么感觉有点怪怪的，突然想到了鲁迅先生笔下的孔乙己说回字有四种写法，那是穿着长衫站着喝酒的唯一一人。说话的当吃瓜群众，怎么就跑题了呢？提个问题强行挽尊，诏安一天要吃掉多少个西瓜呢？ 独居的日子家务并不很多，盗用互联网潮流词语可以说是“轻家务”，洗洗衣服，整理房间也没多少事情。但是呢？莫道昆明池水浅，观鱼胜过富春江。小事情也可以有大作为，于平淡处亦可以见到真志趣，充分挖掘“家务+”的深刻内涵，积极探索“洗衣+音乐”、“整理+音乐”、“家务+锻炼”、“家务+性格养成”等等，成就美好人生。这段有点吹牛。 很晚啦，收尾（END）。 附上丰子恺《闲居》原文 注意：如果有人想将愁写成文章，那一定是假的，我只想吐槽，因为不吐不快乐！……

先生你好啊！这第二十封信我不想谈及其他，只想谈谈你在我心中会是什么样子。要是我说的让你生气，那都是我的不好，你尽管不要理我。 也不知道是谁打翻了上帝的沙漏，往日丝丝如缕，不急不缓的时光突然起了波澜。今年尤其是难过的一年，也是婚姻大事被提及次数最多的一年。当然，很可能接下来的2019年又会是一浪高过一浪。这也不全然是糟心的，至少谈得多，思考的次数和深度也就自然增多和加深，是成长过程中很重要的一部分，古语说：“男大当婚，女大当嫁”，亦同此理。先生，哪一天你真的来我家，我妈妈肯定会很喜欢你。你看她老人家，在没有你一星半点儿消息的时候就如此的急切盼望着你的到来，以至于我每次回家，她都三句不离你。每当看到母亲躁急盼望的神情，作为儿子的我内心并不好受，没能让母亲放心如愿心有不安。窗外的云和月依旧恋恋不舍我这一方窗户，而我却念念着你的归期，假如姻缘天定。 对待婚姻甚至恋爱我们一家人相对比较传统与保守，但又不固步自封，有其进步开明的一面，也有所坚守。爸爸妈妈都是本分的实在人，从小妈妈管我比较多，所以我很多事情都会和妈妈沟通并拿出自己的主意，很开心母亲每次都很支持我的想法。对我的家人我是没有秘密的，因为行而有度，理义为先，用现在的话说就是做事知轻重有分寸，所以我希望先生也是这个样子。 胡适先生是我最崇拜的人之一，这位新文化的缔造者却是旧文化的楷模。在鲁迅、梁启超等纷纷以提倡新文化为借口打破旧式婚姻的时候，胡适这个海龟博士却遵从了旧时的婚约同目不识丁的女子结成连理。对于国家的文明，胡适说只消考察三件事：第一看他们怎样对待孩子；第二看他们如何对待女人；第三看他们如何利用闲暇时间。我觉得用这三点来评价一个人的文明程度也完全适合，对待孩子和女人的问题经过这么多年思想建设相信已经成了普遍的共识，在此不表。我想谈谈这第三点如何利用闲暇时间。 学生时代有同学有老师，不住校的时候可以天天回家和爸妈在一起，到了上中学，上大学，住校了和舍友在一起，一个人独处的时候并不多，那时候也不把孤独当做一回事。大学那一会儿一直想做一个很牛的工程师，几度想自己写操作系统，写编译器，年轻的时候有的是敢做梦的勇气，用现在的话说就是：”要么在自学，要么在去图书馆借书自学的路上“。到现在我已经毕业4年3个月21天，因为家在老家红星，所以我就不得以要自己租房子住，坦诚的说，也会很无聊，也会经常焦虑。但是这些都没把我给困住，反而我看书看得更勤快，偶尔也练练字，练练琴，做做家务，傍晚时候坚持去公园走上2-3公里已经成了我3年来的习惯。经历过风雨更知道有把雨伞的可贵，漫漫的人生路上如果能有1-2个爱好作为消遣，养成健康的生活习惯，这样的人生会更有趣一些，自律从某种程度上是自我的认同，认同了自己人生也就更加的自由，灵魂的自由！ 要写的东西还有很多很多，索性不写了，你要是出现了，我再好好说给你听。 祝君 安好！秋天来了，早晚多添一件衣服。 慕年 写于2018年10月21日 11时 晚 。侨星大厦宿舍。……
